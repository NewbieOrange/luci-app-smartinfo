luci-app-smartinfo
=================================================
VERSION 2.0

简述:
    此脚本程序包含用于1个用于后台监控并记录硬盘健
康状况的脚本，1个用户向LuCI汇报健康状况的脚本以及
一个共用函数脚本。除去脚本外还包含一个LuCI设置页
面以及一个LuCI状态页显示组件。
    此工具用于在基于OpenWRT的路由器上监控USB硬盘
盒中硬盘的监控状况而使用。


测试环境:
    在 OpenWRT 14.07 上开发
    在 OpenWRT 15.05 上测试通过


依赖关系:
    smartmontools
